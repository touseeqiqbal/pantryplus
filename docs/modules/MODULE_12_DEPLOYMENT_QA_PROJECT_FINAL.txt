2. DEPLOYMENT STRATEGY
================================================================================

PantryPlus is designed for continuous deployment and easy maintenance.

DEPLOYMENT OPTIONS:
- Primary: Vercel (recommended for Next.js 14 App Router)
- Secondary: Firebase Hosting, Netlify, or AWS Amplify

STEPS:
1. Connect GitHub repository to Vercel.
2. Configure Environment Variables (.env.local keys).
3. Automatic build triggering on git push.
4. Preview deployments for pull requests.

COMPONENTS:
- `.env.local`: For Firebase API keys.
- `next.config.mjs`: Build and PWA configuration.
- `package.json`: Scripts for build and start.
