1. OVERVIEW
================================================================================

The Task Management module enables households to coordinate and track household
tasks, assign responsibilities to members, set priorities and due dates, and
monitor task completion. This module improves household efficiency and ensures
fair distribution of responsibilities.

KEY FEATURES:
- Task creation and assignment
- Priority levels (low, medium, high)
- Task categories (cleaning, cooking, shopping, maintenance, other)
- Due date tracking
- Task filtering (all, my tasks, completed)
- Task completion tracking
- Recurring tasks support (infrastructure)

4. TECHNICAL IMPLEMENTATION
================================================================================

4.1 TASKS PAGE (app/tasks/page.tsx)
------------------------------------

FEATURES:
- Task list with filtering
- Add task modal
- Complete/delete actions
- Priority and category badges
- Due date display
- Household member assignment

STATE MANAGEMENT:
- showAddModal: Modal visibility
- filter: Current filter (all/my-tasks/completed)
- formData: New task form state
- tasks: Array of tasks

FILTERING LOGIC:
```typescript
const getFilteredTasks = () => {
  if (filter === 'my-tasks') return getMyTasks();
  if (filter === 'completed') 
    return tasks.filter(t => t.status === 'completed');
  return tasks.filter(t => t.status !== 'completed');
};
```

TASK FORM:
- Title input
- Description textarea
- Category dropdown
- Priority dropdown
- Due date picker
- Submit/cancel buttons

4.2 TASKS HOOK (lib/hooks/useTasks.ts)
---------------------------------------

EXPORTED FUNCTIONS:
- addTask(task): Create new task
- updateTask(id, updates): Update task
- completeTask(id): Mark task as complete
- deleteTask(id): Remove task
- getMyTasks(): Get tasks assigned to current user
- getTasks(filters): Get filtered tasks

STATE:
- tasks: Array of tasks
- loading: Loading state

FILTERS:
```typescript
getTasks({
  status?: 'pending' | 'in-progress' | 'completed'
  assignedTo?: string
  category?: string
  priority?: string
})
```

SYNCHRONIZATION:
- Real-time Firebase sync
- Local IndexedDB caching
- Household filtering
- Offline support

5. DATA MODELS
================================================================================

5.1 TASK OBJECT
---------------
{
  id?: number
  firebaseId?: string
  householdId: string
  title: string
  description?: string
  category: 'cleaning' | 'cooking' | 'shopping' | 'maintenance' | 'other'
  assignedTo?: string                 # User ID
  createdBy: string                   # User ID
  priority: 'low' | 'medium' | 'high'
  dueDate?: string                    # ISO date string
  recurring?: {
    frequency: 'daily' | 'weekly' | 'monthly'
    interval: number
  }
  status: 'pending' | 'in-progress' | 'completed'
  completedAt?: string
  completedBy?: string                # User ID
  createdAt: string
  updatedAt: string
  syncStatus: 'synced' | 'pending' | 'error'
}

6. USER WORKFLOWS
================================================================================

6.1 CREATING A TASK
-------------------
1. Navigate to /tasks
2. Click "+ Add Task"
3. Enter task title (e.g., "Clean kitchen")
4. Add description (optional)
5. Select category (cleaning)
6. Set priority (medium)
7. Choose due date (tomorrow)
8. Optionally assign to member
9. Click "Add Task"
10. Task appears in task list

6.2 COMPLETING A TASK
---------------------
1. View task list
2. Find task to complete
3. Click "Complete" button
4. Task marked as completed
5. Task moves to completed filter
6. Completion timestamp recorded
7. Completed by user recorded

6.3 VIEWING MY TASKS
--------------------
1. Navigate to /tasks
2. Click "My Tasks" filter
3. See only tasks assigned to you
4. Tasks sorted by priority and due date
5. Complete or update tasks
6. Track personal task progress

6.4 MANAGING HOUSEHOLD TASKS
-----------------------------
1. View all household tasks
2. Assign unassigned tasks to members
3. Update task priorities
4. Set or adjust due dates
5. Monitor task completion
6. Ensure fair task distribution

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Tasks not appearing
SOLUTION:
- Check household filter
- Verify task was saved
- Refresh the page
- Check Firebase sync status

ISSUE: Cannot complete task
SOLUTION:
- Verify you have permission
- Check internet connection
- Try refreshing page
- Check browser console

ISSUE: Task assignments not working
SOLUTION:
- Verify household members exist
- Check member permissions
- Ensure valid user IDs
- Refresh member list
